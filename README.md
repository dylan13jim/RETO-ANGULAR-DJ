#BACK -JSON-SERVER

Instalacion json-server
```npm i -g json-server```
Ingreso a directorio o carpeta api en back
`cd api`
Comando para ejecutar back
Puerto en el que est trabajando = 3000
`json-server --watch db.json --port 3000`
